Here are some keyboard shortcuts available when a hardware keyboard is present. If you're using a phone where the main input type is a virtual keyboard, please see the VirtualKeyboard help topic.

The following keys are sent unadulterated by ConnectBot:

  • Ctrl
  • Left Shift
  • Alt
  • Tab

The following keys are achieved through alternative means:

  • Escape (ESC)
    • Use the Search key to send ESC.
  • Delete (DEL)
    • Use Left Shift + Backspace to send DEL.
    • Function keys (F1 through F10)
      • Hold down the Right Shift key and press numbers 1 through 10 to send F1 through F10 respectively.
